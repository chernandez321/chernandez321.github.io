### Nivel 0 -> 1

**Objetivo:** Conéctate al servidor usando SSH y encuentra la contraseña en el archivo `readme`.



**Comando:**
```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220

cat readme
```

¡La contraseña para el siguiente nivel es: `bandit0password`!
